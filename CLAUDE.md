# 詠唱メーカー【完成版 v9】

## サービス概要
Twitchチャットと連携して厨二病な詠唱文を生成するWebツール。
配信で使う時は画面をキャプチャして映すだけ。OBSなしでもChromeだけで動く。

**URL:** https://eishou-maker.vercel.app

---

## 実装済み機能

### 基本機能
| 機能 | 内容 |
|------|------|
| Twitch連携 | `!word 単語` でチャットから単語を収集 |
| 単語カード | クリックで選択、ドラッグで並び替え、個別削除 |
| 自由入力 | 配信者が直接単語を追加できる |
| AI詠唱生成 | 選んだ単語をClaudeが厨二病な詠唱文に |
| 自己紹介詠唱 | 言霊モード（単語から）/ プロフィールモード（フォーム入力）切り替え |
| 自動タイプ | AIが単語の内容から最適なタイプを自動判断（デフォルト） |
| レートリミット | Upstash Redisで永続化・確実に機能 |
| Vercelプロキシ | APIキー不要で公開 |

## 設計思想
「普通のWebツールとして誰でも使える。配信で使う時は画面をキャプチャして映すだけ。」
- 1ページ完結設計
- 3カラム儀式祭壇型レイアウト
- OBSなしでもChromeだけで動く
- サブスクライバー特典でマネタイズの足がかり
- 5テーマで配信者の個性を演出
- 配信者がリスナーに説明するための素材（コマンドコピー・画像）を内蔵
